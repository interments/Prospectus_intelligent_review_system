# API与能力采购清单（价格波动模块）

## 1) LLM API（结构化抽取）

必备：
- 中文理解强
- 支持 JSON 输出约束（JSON mode / function calling）
- 支持较长上下文（建议 >= 32k）

建议采购：
- 主模型：1个（成本优先）
- 备模型：1个（可用性兜底）

关键计费项：
- 输入token单价
- 输出token单价
- 并发/QPS限制

## 2) OCR/文档解析 API（关键）

必备：
- PDF转结构化文本
- 表格识别（跨页更佳）
- 返回页码（最好含坐标）

可选增强：
- 标题层级识别
- 阅读顺序重建

## 3) 汇率数据 API/数据源

必备：
- USD/CNY 历史日频数据
- 支持指定日期查询

策略：
- 主数据源：akshare（本地）
- 备用数据源：在线汇率接口（防止节假日/缺失）

## 4) 可选：PDF高质量抽取替代

若OCR仍不稳定，可采购/部署：
- 专门的金融文档解析服务（支持表格+段落）

## 5) 我这边落地时需要你提供

- 各API Key
- Base URL（若有私有网关）
- 模型名白名单
- 预算上限（按月）
- 是否允许国内/海外服务
